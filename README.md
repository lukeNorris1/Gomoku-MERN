


### Env Variables

Create a .env file in then root and add the following

```
NODE_ENV = development
PORT = 5000
MONGO_URI = your mongodb uri

## Install Dependencies

```
npm install
npm install ./frontend
```

### Run

# Run Frontend and backend
npm run dev

```
# Run frontend
npm run client

# Run backend
npm run server

# Run database - empty data
npm run bootstrapdb
```
